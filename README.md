# crawling_py
Web Crawling with python
# Info
* Language
  * Python
* Library
  * Beautifulsoup4 : 웹페이지의 html요소를 파싱하여 필요한 데이터를 추출하기 위해 사용
  * flask : 추출된 데이터를 간단한 웹페이지에 출력하기 위해 사용
  * csv : csv파일로 저장하기 위해 사용
